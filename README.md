🚀 Features
Modern UI/UX: Optimized for Web, Tablet, and Mobile.
Responsiveness: Fully responsive layout designed to adapt to any device.
Fast Development: Powered by Vite for fast builds and efficient development.
Integrated Backend: Laravel backend for API management and database connectivity.
Reusable Components: Built with MUI, SweetAlert2, and FullCalendar for modularity.
Interactive Dashboards: Powered by Recharts for dynamic data visualization.
